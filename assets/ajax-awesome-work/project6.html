React Memory Game
by Connor Lee
React Memory Game is a front end React app that tests the player's short term memory. The game displays 12 different pictures at a time and the Player must select a different picture after each point. Once a picture is selected, the gameboard's state changes and shuffles the displayed images. User's score and high score are saved in state and update on the DOM.

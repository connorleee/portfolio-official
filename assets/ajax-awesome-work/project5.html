Stem
By: Connor Lee, Gabbi Stein, Brandon Lublin, and Vishaal Diwan
Stem is a full-stack React app designed to aggregate your friends' social media content. The full MERN (MongoDB, ExpressJS, React, Node) stack is utilized within an MVC architecture. Users use their Google account to log in using OAuth, web tokens, and cookies for persistence. The website's design uses a combination of the Bootstrap and Materialize CSS libraries.
This application was designed and built by a team of four.

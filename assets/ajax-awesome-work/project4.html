S2ki Web Scraper
By: Connor Lee
S2ki Scraper utilizes a Mongo database with Mongoose models, Handlebars template engine, and Cheerio for a jQuery-like backend experience to scrape the S2ki website's most recent articles. User has the ability to save articles as well as post comments for other users to see.

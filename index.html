About Me

Connor Lee

Full Stack Web Developer

I am a Full Stack Web Developer and graduate from the University of Washington Coding Bootcamp. I have a strong background with React, Node, SQL, NoSQL, REST APIs, and more. My list of known technologies is expanding by the day and I am currently spending time learning testing with Jest.

Previously, I worked with a great team to build an internal tool for a local property management firm (Star Magnolia Apartments, LLC) to help property managers manage leases, finances, maintenance requests, etc. I was heavily involved with the design and architecture of the application and ensuring the product met the client's needs while also planning for future scalability. I also set up the framework of the application's front end with React and was heavily involved with the REST API and protected routes requiring authentication with JSON Web Tokens sent through the HTTP headers. Additionally, I set up the application's user authentication system, log-in session persistance, and application layouts per authorization level.

Most recently, I developed a project management tool (Bug Tracker) in order to help teams track project progress and pull tickets for issues/bugs/etc. The inspiration for this project came because my team for the Appman struggled to find an agile project management software that was simple with a clean UI and minimal bloat features.


Additionally, I have a background as a Civil Engineer graduate from the University of Washington and worked as a heavy civil construction contractor for 4 years. As a contractor, I quickly was drawn to the technology side of the industry and found a passion in softwares and technologies - such as drones in construction - that streamlined projects while also being scaleable for use on future projects. This sparked my interest in software development and led me to the intensive UW Coding Bootcamp that I completed while working full time. My passion for technology and development was solidified as I now possess the skills required to build the technologies I believe can benefit the world.
